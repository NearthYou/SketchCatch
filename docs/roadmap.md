# Roadmap

## Phase 0: Environment Setup

Create the monorepo, tooling, documentation, local PostgreSQL Compose file, and CI.

## Phase 1: Static Architecture Board Prototype

Build a static frontend prototype for arranging AWS architecture resources.

## Phase 2: Prompt-to-Architecture Mock

Create mock prompt-to-architecture flows without using a real AI provider.

## Phase 3: Cost/Risk Rule Engine Mock

Add local mock rules for cost and risk feedback.

## Phase 4: Terraform Generation

Generate Terraform files from approved architecture definitions.

## Phase 5: Controlled AWS Deployment

Deploy approved architectures through controlled backend workflows.

## Phase 6: Auto-Delete Sessions and Reports

Automatically delete practice resources after session expiration and provide user reports.
