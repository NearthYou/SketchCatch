# 고도화 보류 기록

## 2026-07-12 - Workspace 레벨 분리 이후로 미루는 항목

- AI Dock의 최종 소속과 전체 대화 UX 재설계
  - 지금은 Architecture Board와 Deployment Wizard가 동시에 편집 패널로 겹치지 않도록 열림 상태만 정리한다.
  - AI Dock을 Architecture 전용으로 둘지, Deployment 판단 보조까지 맡길지는 두 레벨이 안정된 뒤 결정한다.

- Reverse Engineering 화면 재구성
  - 이번 작업은 일반 Workspace의 Resource/Terraform과 Deployment 위자드 분리가 범위다.

- 새 DB migration
  - 현재 Deployment, Plan 승인, Direct Apply, Git/CI/CD handoff 계약을 재사용한다.

## 2026-07-17 - Reverse Engineering 결과 이해 개선에서 보류하는 항목

- Lambda, ALB, CloudFront, IAM, KMS 등 새 AWS ResourceType의 Terraform 매핑과 import 제안
  - 이번에는 발견·표시·검토 상태만 바로잡는다.
  - Resource별 설정 검증과 Terraform 계약이 준비된 뒤 정식 자동 처리 범위로 올린다.

- AWS 권한 정책, Resource Explorer 활성화, 실제 AWS 조회 범위 확대
  - 이번에는 현재 read-only scan의 결과 표현만 바꾼다.

- 비용·보안 분석 규칙과 Deployment Wizard 승인·실행 흐름 변경
  - 지원 전 Resource는 계속 확정 분석과 배포 대상에서 제외한다.
