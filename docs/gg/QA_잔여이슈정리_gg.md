# QA 잔여 이슈 정리

**기준 문서:** `QA노트 397e63a74db280259fdfd39303df6931.md`의 미완료 항목

이 문서는 현재 제품 QA에서 남은 이슈를 우선순위별로 정리한다. 완료 표시된 항목은 다시 넣지 않는다.

## 범위 제외

아래는 별도 담당 범위이므로 이 문서의 작업·완료 조건에 넣지 않는다.

- CI/CD 자동 연결과 설정 통합
- 보드 내 CI/CD 설정, 배포 이력, Output URL 관리
- Terraform plan/apply, 실제 배포, 배포 후 URL 등록
- 배포된 인프라의 Destroy 및 배포 상태 복구

## P0 — 사용자 데이터와 시작 흐름

### 선택한 템플릿이 프로젝트 생성 단계에서 사라짐

템플릿의 `시작`을 눌러도 프로젝트 생성 화면에서 선택 정보가 유지되지 않아 다시 찾아야 한다.

완료 기준:

- 템플릿 ID와 버전이 생성 화면까지 유지된다.
- 생성 화면에서 선택한 템플릿 이름과 미리보기를 확인할 수 있다.
- 프로젝트 생성 후 선택한 템플릿이 초기 Board에 적용된다.

### Auto Scaling Group 표시 오류

템플릿의 ASG가 메인 Board에서 잘못 보이거나 중복·오배치된 것으로 보인다.

완료 기준:

- ASG 아이콘과 이름이 올바르게 보인다.
- Launch Template, Subnet, Load Balancer 관계가 실제 연결대로 보인다.
- 내부 EC2 표현이 ASG 자체와 중복되지 않는다.

## P1 — 화면 표시 오류

### 의미를 알 수 없는 제목이 노출됨

완료 기준:

- 제목의 데이터 출처를 확인한다.
- 내부 식별자, placeholder, 번역 키가 사용자 화면에 노출되지 않는다.
- 프로젝트 또는 현재 화면을 설명하는 제목으로 대체한다.

### 아이콘이 깨짐

완료 기준:

- 아이콘 경로와 SVG import, 리소스 타입 매핑을 확인한다.
- 미지원 Resource도 깨진 이미지 대신 공통 fallback 아이콘을 보인다.
- 다크 모드에서도 아이콘 대비가 유지된다.

## P2 — 템플릿·모듈 이해

### 모듈에 포함된 Resource를 미리 보기 어려움

모듈 카드나 상세 화면에서 아래 정보를 확인할 수 있게 한다.

- 아키텍처 썸네일
- 포함 Resource 목록과 개수
- 주요 연결 관계
- 지원 Provider
- 필요한 입력값과 제공하는 출력값
- 모듈 또는 템플릿 버전

## 진행 순서

1. 템플릿 선택 상태 유지와 ASG 표시 오류
2. 제목·아이콘 화면 오류
3. 모듈 미리보기 정보

각 항목은 실제 사용 흐름에서 재현한 뒤, 단위 테스트 또는 화면 테스트를 추가하고 해결 여부를 QA 노트에 갱신한다.
