# AI 초안 원형 Orbit 배치

## 목표

- 데스크톱 대화 중 세 공전 궤도를 모두 원으로 표시합니다.
- 궤도, 중심점, AWS 리소스 아이콘이 대화창을 침범하지 않게 합니다.
- 모바일과 최종 Preview 배치는 바꾸지 않습니다.

## 배치

- `960px` 이상에서는 대화창 오른쪽에 장식 전용 영역을 둡니다.
- 장식 전용 영역은 대화창의 오른쪽 경계와 간격 이후부터 시작합니다.
- Orbit의 중심은 장식 전용 영역 안에서 가운데에 둡니다.
- Orbit 크기는 장식 전용 영역의 가로·세로 중 작은 값에 맞춰 정사각형으로 제한합니다.
- 바깥 리소스 아이콘까지 장식 전용 영역 안에 남도록 여백을 확보합니다.

## 공전 표현

- 세 궤도는 서로 다른 지름의 원입니다.
- 타원을 만들던 세로 축소와 기울기 보정은 제거합니다.
- 공전 속도와 방향, option 선택 시 반응, 마지막 수렴 효과는 유지합니다.

## 반응형 경계

- `959px` 이하의 기존 모바일 CSS는 그대로 유지합니다.
- 완성된 Architecture Preview와 적용 흐름은 변경하지 않습니다.

## 검증

- 렌더링 결과에서 세 궤도의 비율이 모두 `1`인지 확인합니다.
- 데스크톱 CSS에서 장식 영역이 대화창 오른쪽부터 시작하는지 확인합니다.
- `1024×768`, `1440×900`에서 대화창과 장식이 겹치지 않는지 브라우저로 확인합니다.
- 기존 공전·반응·수렴 동작과 모바일 배치가 유지되는지 확인합니다.
