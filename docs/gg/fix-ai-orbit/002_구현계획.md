# AI 초안 원형 Orbit 배치 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 데스크톱 AI 초안 대화 중 원형 Orbit을 대화창 오른쪽 전용 영역에 표시합니다.

**Architecture:** Orbit 표현은 기존 `DecorativeAwsOrbit`이 계속 담당합니다. 원형 여부는 컴포넌트의 layer metadata와 CSS transform으로 고정하고, 데스크톱 배치는 기존 전체 화면 stage 안에 대화창 오른쪽 전용 canvas를 두어 분리합니다.

**Tech Stack:** React, TypeScript, CSS Modules, Node test runner, Next.js

## Global Constraints

- `960px` 이상 데스크톱만 변경합니다.
- `959px` 이하 모바일 배치는 유지합니다.
- 공전 속도·방향, option 선택 반응, 마지막 수렴, 최종 Preview는 유지합니다.
- 새 의존성은 추가하지 않습니다.
- 기존 AWS 작업 폴더의 미커밋 파일은 건드리지 않습니다.

- [ ] **Step 1: 원형 궤도 회귀 테스트를 먼저 작성**

  기존 타원 기대값을 다음 원형 계약으로 교체합니다.

  ```tsx
  test("Orbit은 데스크톱 원형 계약과 기존 모바일 타원 정보를 함께 가진다", () => {
    const html = renderToStaticMarkup(
      createElement(DecorativeAwsOrbit, {
        composition: createDecorativeOrbitComposition([]),
        convergenceLevel: 0,
        reactionKey: "initial",
        visibleRingCount: 3
      })
    );

    assert.equal(html.match(/data-desktop-orbit-shape="circle"/g)?.length, 3);
    const aspects = [...html.matchAll(/data-orbit-aspect="([\d.]+)"/g)].map((match) =>
      Number(match[1])
    );
    assert.deepEqual(aspects, [1.25, 1.4, 1.55]);
    assert.match(html, /data-orbit-tilt="-11"/);
    assert.match(html, /data-orbit-tilt="8"/);

    for (const layer of [0, 1, 2]) {
      const layerStart = html.indexOf(`data-layer="${layer}"`);
      const nextLayerStart = html.indexOf(`data-layer="${layer + 1}"`, layerStart);
      const layerMarkup = html.slice(layerStart, nextLayerStart < 0 ? undefined : nextLayerStart);

      assert.ok(layerMarkup.indexOf("data-orbit-ring") < layerMarkup.indexOf("data-orbit-track"));
    }
  });
  ```

- [ ] **Step 3: Orbit markup을 원형 계약으로 단순화**

  함수 설명을 `실제 AWS icon을 데스크톱 원형·모바일 타원 궤도 위에서 장식적으로 공전시킵니다.`로 바꾸고, 현재 layer frame의 속성과 style을 다음처럼 변경합니다.

  ```tsx
  {ORBIT_LAYER_GEOMETRY.map(({ aspect, layer, tilt }) => (
    <span
      className={styles.orbitLayerFrame}
      data-desktop-orbit-shape="circle"
      data-layer={layer}
      data-orbit-aspect={aspect}
      data-orbit-tilt={tilt}
      data-ring-visible={layer < visibleRingCount ? "true" : "false"}
      key={layer}
      style={
        {
          "--orbit-ellipse-tilt": `${tilt}deg`,
          "--orbit-ellipse-y-scale": 1 / aspect,
          "--orbit-ellipse-y-scale-inverse": aspect
        } as CSSProperties
      }
    >
      <span className={styles.orbitRing} data-orbit-ring="true" />
      <span className={styles.orbitLayer} data-orbit-track="true">
        {composition.glyphs
          .map((glyph, index) => ({ glyph, index }))
          .filter(({ glyph }) => glyph.orbitLayer === layer)
          .map(({ glyph, index }) => (
            <span
              className={styles.orbitGlyphPosition}
              data-mobile-hidden={index >= MOBILE_ORBIT_GLYPH_COUNT ? "true" : "false"}
              data-response-glyph={composition.responseGlyphIndex === index ? "true" : "false"}
              key={`${glyph.resourceId}-${index}`}
              style={
                {
                  "--glyph-angle": `${glyph.angle}deg`,
                  "--glyph-scale": glyph.sizeScale
                } as CSSProperties
              }
            >
              <span className={styles.orbitGlyphCounter}>
                <span className={styles.orbitGlyphPlate}>
                  <img alt="" draggable={false} src={glyph.iconUrl} />
                </span>
              </span>
            </span>
          ))}
      </span>
    </span>
  ))}
  ```

  모바일에 필요한 `aspect`와 `tilt` metadata는 유지합니다. 최종 변환값 대신 `--orbit-ellipse-*` 원본값을 전달해 데스크톱 CSS가 원형으로 덮어쓸 수 있게 합니다. Glyph의 `--glyph-angle`, `--glyph-scale`은 유지합니다.

- [ ] **Step 4: 데스크톱에서만 타원 보정을 해제하고 stage를 분리**

  기존 모바일 계산값은 다음 변수 연결로 유지합니다.

  ```css
  .orbitLayerFrame {
    --orbit-ellipse-tilt: 0deg;
    --orbit-ellipse-y-scale: 1;
    --orbit-ellipse-y-scale-inverse: 1;
    --orbit-kick: 18deg;
    --orbit-tilt: var(--orbit-ellipse-tilt);
    --orbit-y-scale: var(--orbit-ellipse-y-scale);
    --orbit-y-scale-inverse: var(--orbit-ellipse-y-scale-inverse);
  }
  ```

  대화창 오른쪽 전용 영역은 데스크톱 media query로만 추가합니다.

  ```css
  .splitLayoutConversation {
    --orbit-stage-left: calc(
      var(--conversation-dock-left) + var(--conversation-dock-width) +
        var(--conversation-side-gap)
    );
  }

  @media (min-width: 960px) {
    .orbitLayerFrame {
      --orbit-tilt: 0deg;
      --orbit-y-scale: 1;
      --orbit-y-scale-inverse: 1;
    }

    .stageCanvas {
      bottom: 0;
      left: var(--orbit-stage-left);
      overflow: hidden;
      position: absolute;
      right: 0;
      top: 0;
    }

    .orbitScene {
      left: 50%;
      width: min(calc(100% - 96px), 650px, calc(100vh - 190px));
    }
  }
  ```

  기존 공전 keyframe과 `@media (max-width: 959px)` 블록은 수정하지 않습니다.
