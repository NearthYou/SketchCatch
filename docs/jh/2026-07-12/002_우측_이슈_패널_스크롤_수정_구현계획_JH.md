# 우측 이슈 패널 스크롤 수정 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 우측 코드 패널의 Architecture와 Terraform 진단을 하나의 세로 스크롤로 끝까지 탐색할 수 있게 한다.

**Architecture:** `WorkspaceIssuesPanel`이 두 진단 그룹을 자연 높이로 쌓고 유일한 세로 스크롤을 소유한다. 기존 진단 컴포넌트, 상태, 코드/Issues 리사이즈 경계는 유지하고 CSS 레이아웃 계약만 회귀 테스트로 고정한다.

**Tech Stack:** Next.js 16, React 19, CSS Modules, Node.js `node:test`, TypeScript

## Global Constraints

- 현재 브랜치에서 작업한다.
- 컴포넌트 구조, 진단 상태, API 계약을 변경하지 않는다.
- 코드/Issues 리사이즈 기능과 각 진단 카드 버튼 동작을 유지한다.
- 새 런타임 의존성을 추가하지 않는다.

### Task 1: 통합 Issues 스크롤 회귀 수정

**Files:**
- Modify: `apps/web/features/workspace/workspace-right-panel-layout.test.ts`
- Modify: `apps/web/features/workspace/WorkspaceIssuesPanel.module.css`

**Interfaces:**
- Consumes: `WorkspaceIssuesPanel`의 CSS Module 클래스 `issuesPanel`, `terraformIssues`
- Produces: Architecture와 Terraform 진단 전체를 감싸는 단일 세로 스크롤 레이아웃 계약

- [ ] **Step 1: 실패하는 레이아웃 회귀 테스트 작성**

`workspace-right-panel-layout.test.ts`의 CSS 소스 상수에 다음을 추가한다.

```ts
const workspaceIssuesStylesSource = readWorkspaceFile("WorkspaceIssuesPanel.module.css");
```

다음 테스트를 추가한다.

```ts
test("combined architecture and terraform issues share one reachable vertical scroll", () => {
  const combinedIssuesRule = getCssRule(workspaceIssuesStylesSource, "issuesPanel");
  const terraformIssuesRule = getCssRule(workspaceIssuesStylesSource, "terraformIssues");

  assert.match(combinedIssuesRule, /\bgrid-template-rows:\s*auto auto;/);
  assert.match(combinedIssuesRule, /\boverflow-y:\s*auto;/);
  assert.match(combinedIssuesRule, /\bscrollbar-gutter:\s*stable;/);
  assert.doesNotMatch(combinedIssuesRule, /\boverflow:\s*hidden;/);
  assert.doesNotMatch(terraformIssuesRule, /\boverflow:\s*hidden;/);
});
```

.issuesPanel {
  align-content: start;
  display: grid;
  grid-template-rows: auto auto;
  height: 100%;
  min-height: 0;
  overflow-y: auto;
  scrollbar-gutter: stable;
}

.terraformIssues {
  min-height: 0;
}
