# Task 1 Report - 안전 기본값과 확장 리소스 카탈로그

## 변경 내용

- 신규 resource node에만 `ResourceItem.id` 기반의 안전 기본값 whitelist를 적용했다. 기존 node panel 열기와 ASG `desiredCapacity`, RDS read replica에는 값을 backfill하지 않는다.
- `AUTO_SCALING_POLICY`, `aws-autoscaling-policy`, palette 항목과 EC2 Auto Scaling resource icon을 추가했다.
- ASG, Security Group, Listener, CloudWatch Alarm, Autoscaling Policy의 main/core parameter와 reference/nested block catalog를 추가했다.
- catalog generator가 override의 `core`를 generated catalog에 보존하도록 수정하고 `pnpm catalog:generate`를 실행했다.
