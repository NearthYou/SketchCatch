# Workspace AI 작업을 AI 채팅으로 통합

Workspace 오른쪽 패널에 분산된 설계 제안, 오류 분석, 에이전트 리뷰 실행 버튼은 입력 문맥과 AI 결과 상태의 소유 위치를 갈라놓고, 새로고침 후 대화와 수정 가능 상태가 어긋나게 만들었다. 세 작업의 실행과 결과는 AI 채팅이 소유하고 오른쪽 패널은 현재 Resource, Terraform 코드, Terraform Issue 문맥만 제공한다. 데스크톱 AI 채팅은 Architecture Board와 동시에 조작 가능한 비모달 플로팅 패널로 두고, 모바일에서는 전체 화면 모달을 유지한다. 오류별 수정과 전체 수정은 모두 명시적 사용자 승인 뒤에만 실행하며, 전체 수정은 모든 오류 분석 결과가 현재 Terraform fingerprint와 일치할 때 원자적으로 적용한다.
