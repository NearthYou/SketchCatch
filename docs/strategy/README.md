## 현재 결론

SketchCatch는 단순한 그림판형 클라우드 도구가 아니라, AWS 입문자가 자연어로 아키텍처를 만들고, 시각적으로 검토하고, IaC로 변환하며, 비용과 위험을 확인한 뒤 안전하게 실습 배포할 수 있게 만드는 학습형 IaC 플랫폼을 목표로 합니다.

현재 저장소는 제품 기능 구현 전 단계의 개발 환경, 배포 기반, RDS/S3 저장 기반, CI/CD 기반까지 준비된 상태입니다.
