# 자체 호스팅 글꼴 자산

- `pretendard-variable/`: 서비스 전역 UI에 사용하는 Pretendard Variable `woff2` 파일과 라이선스입니다.
- `line-seed/`: 대체 비교용으로 보관하는 LINE Seed Sans KR Thin, Regular, Bold `woff2` 파일과 공식 라이선스 고지 및 SIL Open Font License 1.1입니다.
- `space-grotesk/`: 대체 비교용으로 보관하는 Space Grotesk `woff2` 파일입니다.
- `jetbrains-mono/`: 대체 비교용으로 보관하는 JetBrains Mono `woff2` 파일입니다.

Pretendard를 전역에 적용합니다. 나머지 글꼴은 현재 사용하지 않는 대체 자산입니다.
