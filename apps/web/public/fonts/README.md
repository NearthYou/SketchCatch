# 자체 호스팅 글꼴 자산

- `line-seed/`: 서비스 전역 UI에 사용하는 LINE Seed Sans KR Thin, Regular, Bold `woff2` 파일과 공식 라이선스 고지 및 SIL Open Font License 1.1입니다.
- `space-grotesk/`: 로컬 호스팅용으로 Google Fonts에서 받은 Space Grotesk `woff2` 파일입니다.
- `jetbrains-mono/`: 로컬 호스팅용으로 Google Fonts에서 받은 JetBrains Mono `woff2` 파일입니다.

LINE Seed Sans KR는 전역에 적용됩니다. 나머지 글꼴은 현재 사용하지 않는 대체 자산입니다.
