# Teaching Notes

- 사용자는 AWS 용어보다 SketchCatch 화면과 실제 동작을 기준으로 설명할 때 이해가 빠르다.
- 어려운 용어를 먼저 나열하지 말고 `파일 보관함 → 공사 관리표 → 출입증` 비유를 먼저 사용한다.
- 한 번에 하나의 관계를 설명하고 마지막에 짧은 확인 문제를 낸다.
- `S3`라는 이름이 보여도 Template 저장, Terraform 기록 저장, 사용자 서비스 Bucket을 서로 같은 것으로 취급하지 않는다.
- SSO 설명에서는 `사람이 Console에 들어갈 때 쓰는 Role`과 `SketchCatch 서비스가 가져오기·배포에 쓰는 Role`을 반드시 분리한다.
